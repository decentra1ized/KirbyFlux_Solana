
# 블록체인실무응용 - KirbyFlux

주제: 분산된 자산을 한 곳에 모아주는 솔루션(멀티워렛 트렌스퍼) 

## Installation

```bash
npm install
# or
yarn install
```

## Build and Run

Next, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
